## RUBRIC: Q024 - Ras-G12V Cancer (14 marks)

### Understanding Ras Mutation (6 marks)

**The normal Ras-GTPase cycle**:
- [ ] Normal Ras: GTP-bound = active (signal ON) ✓ = 1m
- [ ] GAP (GTPase activating protein) recognizes normal Ras ✓ = 0.5m
- [ ] GAP catalyzes GTP→GDP conversion ✓ = 0.5m
- [ ] Ras-GDP = inactive (signal OFF) ✓ = 0.5m

**G12V mutation specifics**:
- [ ] G12V is at position 12, substitutes Valine for Glycine ✓ = 0.5m
- [ ] **MUTATION SITE = GAP RECOGNITION SITE** ✓ = 1.5m  ← KEY!
- [ ] GAP CANNOT RECOGNIZE mutant Ras ✓ = 1m
- [ ] Result: Ras-GTP CANNOT HYDROLYZE to GDP (stuck GTP) ✓ = 1m

### Molecular Consequences (4 marks)

- [ ] Ras-G12V constitutively active (always ON, no OFF switch) ✓ = 1m
- [ ] Constantly recruits Raf kinase (MAPKKK) ✓ = 1m
- [ ] Continuous phosphorylation cascade: Raf→MEK→ERK ✓ = 1m
- [ ] ERK continuously active WITHOUT external growth factor signal ✓ = 1m

### Cancer Mechanism & Clinical Context (3 marks)

- [ ] Uncontrolled cell division (proliferation signal always ON) ✓ = 1m
- [ ] Why normal cells don't divide: Need growth factor to activate Ras
  - With G12V: No growth factor needed ✓ = 1m
- [ ] Real example: Ras-G12V found in pancreatic cancer, lung cancer (~30% of all cancers) ✓ = 1m

### Why More GAP Won't Help (1 mark)

- [ ] Recognition site mutated → more GAP won't bind → won't help ✓ = 1m
- **Common incorrect answer**: "Because there's too much GTP" ❌
- **Correct reason**: "Because the RECOGNITION SITE is mutated, not the catalytic site"

## RUBRIC: Q040 - TGF-β SMAD Pathway (14 marks)

### Canonical Pathway (8 marks)

**Receptor activation**:
- [ ] TGF-β binds TβRII (constitutively active) ✓ = 1m
- [ ] TβRII recruits & phosphorylates TβRI (kinase domain activated) ✓ = 1m

**SMAD activation**:
- [ ] TβRI phosphorylates SMAD2/3 at Ser/Thr residues ✓ = 1m
- [ ] pSMAD2/3 + SMAD4 form heterotrimer ✓ = 1m
- [ ] SMAD complex accumulates in nucleus ✓ = 1m

**Gene regulation**:
- [ ] SMAD complex binds SMAD-binding elements (SBE) in promoter ✓ = 1m
- [ ] Works with TCF/LEF cofactors ✓ = 0.5m
- [ ] Activates p15/p21 (CDK inhibitors) → cell cycle STOP ✓ = 0.5m

### Dual-Role Cancer Context (4 marks)

**Early cancer (normal cells)**:
- [ ] TGF-β stops cell division (tumor SUPPRESSOR role) ✓ = 1m
- [ ] Normal SMAD2/3 and SMAD4 present ✓ = 0.5m

**Late cancer (SMAD4 deleted)**:
- [ ] Loss of SMAD4 breaks canonical pathway ✓ = 1m
- [ ] Non-canonical arms activate (PI3K, Rho, PAR1) ✓ = 0.5m
- [ ] Result: EMT progression, metastasis (tumor PROMOTER role) ✓ = 1m

### TGF-β vs BMP-2 Distinction (2 marks)

- [ ] TGF-β uses pSMAD2/3 + SMAD4 ✓ = 1m
- [ ] BMP-2 uses pSMAD1/5/8 + SMAD4 (different R-SMADs!) ✓ = 1m
- **Concept**: Same SMAD4, different R-SMADs = pathway selectivity

## RUBRIC: Q051 - ER Stress UPR Response (14 marks)

### BiP Sensing Mechanism (3 marks)

- [ ] BiP normally bound to PERK, IRE1, ATF6 (all inhibited) ✓ = 1m
- [ ] ER stress: Misfolded proteins accumulate ✓ = 0.5m
- [ ] BiP leaves regulatory proteins to help misfolds (NOT physically forced) ✓ = 1m
- [ ] PERK, IRE1, ATF6 become FREE & activate ✓ = 0.5m

### Three UPR Branches (9 marks: 3 marks each)

**PERK branch**:
- [ ] PERK autophosphorylates, becomes active ✓ = 0.5m
- [ ] PERK phosphorylates eIF2α (initiation factor) ✓ = 1m
- [ ] Global translation blocked (saves resources) ✓ = 1m
- [ ] BUT ATF4 specifically translated (uORF mechanism allows bypass) ✓ = 0.5m
- [ ] ATF4 makes CHOP (apoptosis factor if stress too long) ✓ = 0.5m

**IRE1 branch**:
- [ ] IRE1 autophosphorylates, becomes active ✓ = 1m
- [ ] IRE1 splices XBP1 mRNA (specific recognition) ✓ = 1m
- [ ] XBP1s (spliced) = active transcription factor ✓ = 0.5m
- [ ] XBP1s makes ER chaperones (BiP, GRP94, PDI) - protein folding recovery ✓ = 1m

**ATF6 branch**:
- [ ] ATF6 protein pre-existing in Golgi (not made new) ✓ = 1m
- [ ] Upon stress: ATF6 cleaved by proteases ✓ = 1m
- [ ] pATF6 (active fragment) enters nucleus ✓ = 0.5m
- [ ] Makes CHOP + ER chaperones ✓ = 0.5m

### Integrated Response & Outcome (2 marks)

- [ ] If stress resolved: Chaperones recover ER, cells survive ✓ = 1m
- [ ] If stress prolonged: CHOP-mediated apoptosis (cell death) ✓ = 1m
